### How do I get install? ###

* Copy file ".vimrc" into path "~/."
* Copy winmanager.zip to the folder ~/.vim/, and unzip it, you will see plugins and doc (please check https://www.vim.org/scripts/script.php?script_id=95)
* Restart vim
